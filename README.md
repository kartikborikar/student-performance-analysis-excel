# 📊 Student Performance - Exploratory Data Analysis Dashboard

## 📝 Project Overview
This project focuses on conducting a thorough **Exploratory Data Analysis (EDA)** on student demographic and academic data using **"Student Performace-Exploratory Data Analysis Dashboard.xlsx"**. The main goal is to uncover hidden insights behind student grades, understand how different factors affect performance, and present these findings through an interactive and dynamic executive dashboard.

---

## 🎯 Key Objectives & Insights
The analysis is structured around several critical metrics found within the dataset to answer key performance questions:
* **Academic Scores:** Analyzing performance and trends across core subjects: `math_score`, `science_score`, and `english_score`.
* **Attendance Impact:** Evaluating `attendance_percent` to see its direct correlation with student success and grades.
* **Demographic & Social Factors:** Segmenting and filtering data by `gender`, `grade_level` (Grade 10, 11, 12), `study_group` (Group A, B, C), and `extra_curricular` activities (Sports, Music, etc.) to find hidden patterns.

---

## 🛠️ Tech Stack & Tools Used
* **Microsoft Excel:** Core tool used for data cleaning, analysis, and visualization.
* **Pivot Tables & Pivot Charts:** Used for backend data aggregation and building the visual matrix.
* **Dynamic Slicers:** Implemented for interactive filtering by Gender, Grade Level, and Study Groups.
* **Advanced Excel Formulas:** Used for structuring and preparing raw data.

---

## 📁 Repository Structure
The project file contains three dedicated sheets:
1. `DATA` - The main database containing raw, cleaned student records (IDs, scores, attendance, and demographics).
2. `Pivot Tables` - The backend processing engine where all data summarization and charts logic reside.
3. `Dashboard` - The final interactive visual interface containing KPI cards, charts, and slicers.

## 💡 How to Explore the Project
1. Download or clone this repository.
2. Open **"Student Performace-Exploratory Data Analysis Dashboard.xlsx"** in Microsoft Excel.
3. Go to the `Dashboard` tab.
4. Click on the **Slicers/Filters** on the side to interactively filter the data and watch the charts update in real-time!
